## 📌 Project Overview

TraiteMate is a full-stack web application that predicts personality traits based on user inputs.
It analyzes responses and generates meaningful personality insights through an interactive interface.

---

## 🎯 Objective

* Analyze user behavior using input data
* Predict personality traits intelligently
* Provide a user-friendly interface
* Demonstrate full-stack (MERN) development

---

## 🚀 Features

* 🔍 Personality Prediction System
* 👤 User Authentication (Login/Signup)
* 📊 Data Analysis & Result Generation
* ⚡ Responsive UI
* 🔐 Secure Backend APIs

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/RishabhAhuja01/TraiteMate.git
cd TraiteMate
```

---

### 2️⃣ Install Dependencies

```bash
# Backend
cd backend
npm install

# Frontend
cd ../frontend
npm install
```

---

### 3️⃣ Environment Variables

Create a `.env` file inside backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

---

### 4️⃣ Run the Project

```bash
# Start Backend
cd backend
npm start

# Start Frontend
cd ../frontend
npm start
```

---

## 📊 How It Works

1. User enters input data
2. Backend processes the data
3. Personality traits are predicted
4. Results are displayed on UI
